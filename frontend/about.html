About Project

MenuMind makes restaurant menus easier to understand.

MenuMind is an AI-powered food-tech web app that scans restaurant menu photos and gives estimated calories, allergen alerts, and healthier alternatives.

🎯

Problem

Restaurant menus usually do not show nutrition details, so users cannot easily know calories, allergens, or healthier options.

💡

Solution

MenuMind reads menu images using OCR and uses AI to suggest food insights in a simple and user-friendly way.

🇮🇳

Indian Food Focus

The project is designed for Indian dishes like dal, sabzi, biryani, paneer, dosa, chole, and thali items.

Technology Used

HTML CSS JavaScript Node.js Express.js OCR Gemini API Nutrition API LocalStorage

Future Scope

✅ Real OCR menu text extraction
✅ Gemini AI based nutrition explanation
✅ Nutrition database API integration
✅ Personalized diet preferences
✅ Zomato / restaurant menu integration idea

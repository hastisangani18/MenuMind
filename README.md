# MenuMind 🍽️

MenuMind is an AI-powered web application that helps users understand restaurant menus more efficiently. Users can upload a menu image, extract text using OCR technology, and receive intelligent food recommendations powered by Gemini AI.

## Features
- Upload menu images
- OCR-based text extraction
- AI-powered food recommendations
- Clean and responsive interface
- Fast menu analysis

## Tech Stack
- HTML
- CSS
- JavaScript
- Tesseract.js (OCR)
- Gemini AI API

## How It Works
1. Upload a restaurant menu image.
2. OCR extracts the menu text.
3. Gemini AI analyzes the extracted content.
4. Personalized food recommendations are displayed.

## Future Improvements
- Nutrition and calorie estimation
- Multi-language menu support

# 🍽️ MenuMind

MenuMind is an AI-powered web application that helps users analyze restaurant menus intelligently. Users can upload a menu image, extract text using OCR technology, and receive smart food recommendations powered by Gemini AI.

## 🚀 Live Demo

🔗 https://menumind-25.netlify.app/

## 📂 GitHub Repository

🔗 https://github.com/hastisangani18/MenuMind

## ✨ Features

* Upload menu images
* OCR-based text extraction using Tesseract.js
* AI-powered food recommendations with Gemini AI
* Fast and responsive user interface
* User-friendly workflow
* Real-time menu analysis

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Tesseract.js (OCR)
* Gemini AI API
* Git & GitHub
* Netlify

## ⚙️ How It Works

1. Upload a restaurant menu image.
2. OCR extracts text from the image.
3. Extracted text is sent for AI analysis.
4. Gemini AI generates food recommendations.
5. Results are displayed in an easy-to-read format.
